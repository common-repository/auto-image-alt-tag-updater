Auto Image Alt Tag Updater automatically updates the alt tags of images in your WordPress posts and pages using the SEO title.

== Description ==

Auto Image Alt Tag Updater is a simple and effective plugin that improves your website's SEO by automatically updating the alt tags of images based on the SEO title of each page or post. Alt tags are essential for search engine optimization, and having them optimized can help increase your website's visibility and ranking on search engines.

Features:
- Automatically updates alt tags for all images in posts and pages.
- Uses the SEO title for generating meaningful alt tags.
- Improves image SEO and accessibility.
- Easy to use with no complex settings.

== Installation ==

1. Download the `auto-image-alt-tag-updater.zip` file.
2. Go to the WordPress admin dashboard.
3. Navigate to **Plugins > Add New**.
4. Click on **Upload Plugin** and select the downloaded `auto-image-alt-tag-updater.zip` file.
5. Click **Install Now** and then **Activate** the plugin.

== Usage ==

Once activated, the plugin will automatically update the alt tags of all images in your posts and pages based on the SEO title. There are no settings to configureâ€”simply install and activate the plugin, and it will do the rest!

== Frequently Asked Questions ==

= Does this plugin work with all themes? =

Yes, the Auto Image Alt Tag Updater works with any WordPress theme.

= Will this plugin affect my existing alt tags? =

Yes, the plugin will overwrite existing alt tags with the SEO title of the page or post.

= Can I revert the changes made by the plugin? =

Currently, there is no built-in option to revert the changes. However, you can manually update the alt tags if needed.
